# 🐍 Snakened (S)

**A linguagem que une a simplicidade de Python com a performance de C.**

Snakened é uma "Linguagem em S" desenhada para ser intuitiva, mas com uma execução ultra-rápida através de compilação nativa.

## 🚀 Objetivo do Projeto
- Sintaxe limpa (estilo Python).
- Velocidade de execução (estilo C).
- Extensão de ficheiro: `.s`
